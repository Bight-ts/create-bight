# Docs Example Sources

This directory contains the tracked source snippets used by the public docs rewrite.

## Layout

- `scaffold/`
  current scaffold-shaped files used in Start Here pages
- `guides/`
  focused examples used in Learn and Build pages
- `reference/`
  tiny imports and usage sketches used by package reference pages

## Rules

- keep examples small
- prefer one-file examples unless the topic needs two files
- mirror real public APIs
- update these files before updating the page content that cites them
